# Packet Sniffer

A Python packet sniffer that captures and analyzes live network traffic.

## Features

* Captures network packets
* Displays source and destination IP addresses
* Identifies TCP, UDP, and ICMP protocols
* Shows basic packet information

## Technologies
* Python
* Scapy
* Npcap

## What I Learned

* How network traffic travels between devices
* The basics of TCP, UDP, and ICMP protocols
* How to capture and analyze packets using Python
* How packet sniffing can be used to monitor network activity
